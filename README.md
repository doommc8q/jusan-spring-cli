## Solid Bank App 
### Running:
    To run download jusan-spring-cli.jar and paste command: 
`java -jar jusan-spring-cli.jar`
### Welcome to CLI bank service.
                Operations:
                1 - show accounts
                2 - create account
                3 - deposit
                4 - withdraw
                5 - transfer
                6 - this message
                7 - exit
